ImpactBridge

Application web responsive pour une organisation solidaire, avec site public, API documentée et espace d’administration sécurisé. Le projet est déployé côté front-end et côté back-end, avec gestion des activités, formulaire de contact, envoi d’emails et tableau de bord d’administration.

Voir le site en ligne Voir la documentation API

Vidéo 1 • Parcours public

Présentation du site public : landing page, navigation responsive, listing des activités, détail d’activité, services et formulaire de contact connecté au backend.

Vidéo 2 • Administration

Démonstration du back-office : Swagger, tableau de bord, réception des messages de contact, changement de statut, création d’activité avec upload d’image, brouillon puis publication, et impact immédiat sur l’affichage côté public.

Stack technique

Le projet repose sur une séparation claire entre interface et API, avec une architecture monorepo, une base PostgreSQL, une authentification JWT, des validations strictes, une documentation Swagger et un système d’upload d’images. Le formulaire de contact est relié à un vrai service d’envoi d’emails.

Next.js TypeScript Express.js Prisma PostgreSQL / Neon JWT React Query Zustand Swagger Cloudinary Zod Tailwind CSS

Application responsive

L’interface a été pensée pour desktop et mobile, autant sur la partie publique que sur l’espace d’administration.

Front-end et back-end déployés

Le site public et l’API sont accessibles en ligne, avec une vraie communication entre le front, la base de données et les services externes.

Gestion de publication

Une activité créée en brouillon reste invisible sur le site public, puis apparaît dès qu’elle est publiée depuis l’admin.

Messages exploitables

Les messages envoyés depuis le formulaire de contact sont stockés, consultables dans l’admin et peuvent être marqués comme lus ou archivés.

Note de démonstration : la vidéo d’administration montre volontairement le cycle complet de gestion d’une activité, depuis sa création en brouillon jusqu’à sa publication et son affichage côté public.
