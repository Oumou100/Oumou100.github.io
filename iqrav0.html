Démonstration de la Plateforme d'Apprentissage

À propos de la plateforme

Cette plateforme d'apprentissage en ligne offre une expérience éducative interactive et personnalisée. Elle permet aux utilisateurs d'accéder facilement à des contenus pédagogiques et de suivre leur progression.

Application Mobile : Coran & Outils Islamiques

Une version mobile a également été développée pour accompagner les utilisateurs au quotidien avec des fonctionnalités utiles et pratiques.

  • 🕌 Horaires de prières en temps réel
  • 🔊 Adhan (appel à la prière)
  • 🧭 Direction de la Qibla
  • 📖 Lecture du Coran
  • 📱 Interface optimisée pour mobile

Technologies utilisées

Laravel / MySQL React Native CLI Next.js Tailwind CSS Git / GitHub
