Description du Projet

RiskLoom.AI est une application web intelligente d'évaluation du risque de crédit utilisant le Machine Learning. Le système analyse le profil financier des demandeurs (âge, emploi, comptes bancaires, montant demandé) et prédit la probabilité d'approbation grâce à un modèle XGBoost entraîné sur des données bancaires.

L'application offre une interface intuitive avec des visualisations interactives (jauge de probabilité, graphiques de risque) et permet la génération de rapports CSV téléchargeables. Elle supporte également plusieurs devises (EUR, MAD, CFA) pour s'adapter aux différents marchés.

Machine Learning Pipeline

  • Modèle : XGBoost Classifier optimisé pour la classification binaire
  • Features : Âge, situation d'emploi, état des comptes, montant et durée du crédit
  • Preprocessing : Encodage des variables catégorielles avec Scikit-learn
  • Évaluation : Probabilité d'approbation en pourcentage avec analyse des facteurs
